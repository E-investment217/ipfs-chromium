using ipfs::Priority = typedef std::uint_least16_t

The urgency of a gateway request.

Determines how many gateways should be involved, and how burdened a gateway should be before not also taking this one on concurrently. Zero is a special value that indicates the block isn't actually required now, but rather might be required soonish (prefetch). There are some cases of special handling for that.
